Running the program:
1.	Ensure python is installed
2.	Install the required libraries matplotlib and pandas using “pip install pandas matplotlib”
3.	Place the dataset file (imdb_top_1000.csv) in the same folder as the program.
4.	Run the python script as a file or as a task.
5.	Select the analysis you require from the terminal menu using the options provided.
6.	A window will open with the visual requested.
7.	Close the visualisation to show another one.
8.	Exit the program when done using options provided.
